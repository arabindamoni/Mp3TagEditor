Mp3tageditor

A light weight mp3 tag editor tool

Download as .zip Download as .tar.gz View on GitHub

Features:

  1. Support for ID3v1 tags.
  2. Support for ID3v2 tags.
  3. User can change album image.
  4. Very lightweight.
  5. Portable (no need to install). https://sourceforge.net/projects/jmp3tageditor/
